## Opdracht
Schrijf een programma die om de naam van de gebruiker vraagt, en vervolgens de gebruiker begroet met `Hallo,` en hun eigen naam.

## Voorbeeld 1
### Input
```console?lang=python
Aïsha
```

### Output
```console?lang=python
Hallo, Aïsha
```
